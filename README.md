# Seguros Saga - Página lista para Vercel

Este proyecto está preparado para publicarse en Vercel con React + Vite.

## Archivos importantes

- `src/App.jsx`: contenido principal de la página.
- `src/styles.css`: estilos visuales.
- `public/LOGO.jpg`: logo de Seguros Saga.
- `package.json`: scripts para instalar, probar y compilar.

## Cómo probar localmente

```bash
npm install
npm run dev
```

## Cómo publicar en Vercel

1. Sube esta carpeta a un repositorio de GitHub.
2. En Vercel, entra a **Add New > Project**.
3. Importa el repositorio.
4. Vercel detectará Vite automáticamente.
5. Usa esta configuración si Vercel la pide:

```text
Framework Preset: Vite
Build Command: npm run build
Output Directory: dist
Install Command: npm install
```

6. Haz clic en **Deploy**.
